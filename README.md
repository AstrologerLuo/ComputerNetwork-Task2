# ComputerNetwork-Task2
这是计算机网络课程的课程设计。
## 开始
### 运行环境
Python(>=3.x)
### 安装
```bash
git clone https://github.com/AstrologerLuo/ComputerNetwork-Task2
```
### 文件环境：
文档1的客户端需要一个名为'send.txt'的文本文件作为输入
### 端口配置：
udpServer.py的服务器监听端口需要自行输入
udpClient.py的客户端需要输入服务器IP和端口
### 程序配置：
updClient.py需要用户输入：
服务器IP地址
服务器端口号
udpServer.py需要用户输入：
监听的端口号
